Hey, du!

Diese Anwendung greift auf deinen Google-Kalender zu und benötigt daher deine Einwilligung um auf deinen Kalender gewähren.

Wenn du dich das erste mal anmeldest, wird sich ein Popup-Fenster öffnen, sobald du auf den Login-Button klickst. Bitte bestätige dort den Zugriff auf deinen Kalender.

Falls kein Popup kommt, hat der Popup-Blocker dies verhindert. Klicke auf das Icon in der Browserleiste (oben), um Popups vorübergehend zuzulassen.
