Diese Anwendung greift auf deinen Google-Kalender zu und benötigt daher deine Einwilligung.

Beim ersten Anmelden wird sich daher ein Popup-Fenster öffnen, sobald du auf den Login-Button klickst. Bitte bestätige dort den Zugriff auf deinen Kalender.

Falls kein Popup kommt, hat der Popup-Blocker dies verhindert. Klicke auf das Icon in der Browserleiste (oben), um Popups vorübergehend zuzulassen.
